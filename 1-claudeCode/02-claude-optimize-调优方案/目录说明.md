## 功能一：Claude VSCode 插件安装

**目录**：`01-claude-vscode-plugin\`

**用途**：安装和配置 Claude Code 的 VSCode / Trae 插件版本

**包含文件**：
| 文件 | 说明 |
|------|------|
| `claude-code-bailian-trae-setup.md` | 百炼 Trae 平台接入指南 |

**适用场景**：
- 初次安装 Claude Code VSCode / Trae 插件
- 了解百炼 API 接入方法

---

## 功能二：Claude + CC-Switch 安装

**目录**：`02-claude-ccswitch\`

**用途**：安装 Claude Code CLI + CC-Switch 双版本共存方案

**包含文件**：
| 文件 | 说明 |
|------|------|
| `claude-code-cc-switch-reinstall-guide.md` | CC-Switch 安装与配置指南 |

**适用场景**：
- 安装 Claude Code CLI 命令行版本
- 配置 CC-Switch 实现多模型路由切换
- 从 VSCode 插件迁移到 CLI 版本

**配套脚本**：`03-claude-optimize\scripts\reinstall-claude-to-cc-switch.ps1`

## 使用流程

### 第一步：初次安装
1. 阅读 `01-claude-vscode-plugin\` 中的指南，完成插件安装
2. 如需 CLI 版本，阅读 `02-claude-ccswitch\` 中的指南，安装 CC-Switch

### 第二步：环境调优
1. 阅读 `03-claude-optimize\claude调优方案.md`，了解完整调优方案
2. 运行一键部署脚本：
   ```powershell
   03-claude-optimize\scripts\setup-claude.ps1 -Phase all
   ```
3. 按需调整配置

### 第三步：新项目复用
1. 将 `03-claude-optimize\.claude\` 目录复制到新项目根目录
2. 将 `03-claude-optimize\templates\CLAUDE.md` 复制为项目的 `CLAUDE.md`，根据实际情况修改
3. 如需 MCP 服务，将 `03-claude-optimize\templates\.mcp.json` 复制到项目根目录
4. 根据项目技术栈，从 `03-claude-optimize\templates\.claude\rules\` 按需拷贝专用规则到项目的 `.claude\rules\` 目录
   ```powershell
   # 示例：Vue 3 + TypeScript 项目
   Copy-Item "03-claude-optimize\templates\.claude\rules\vue3.md" "D:\my-app\.claude\rules\"
   Copy-Item "03-claude-optimize\templates\.claude\rules\typescript.md" "D:\my-app\.claude\rules\"
   ```

## 注意事项

1. **路径问题**：Windows 下路径使用反斜杠 `\`，PowerShell 脚本中注意转义
2. **权限问题**：部署脚本可能需要管理员权限执行（`Set-ExecutionPolicy`）
3. **环境变量**：部分工具（如 MCP 的 GitHub）需要设置对应的 Token 环境变量
4. **版本兼容**：Claude Code 和 CC-Switch 版本需匹配，详见各安装指南
5. **配置层级**：全局配置（`~/.claude/`）对所有项目生效，项目配置（`项目/.claude/`）仅对当前项目生效，两者冲突时项目级优先
